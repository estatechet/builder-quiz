# 브랜드 로고 폴더

실제 브랜드 로고 PNG 파일을 이 폴더에 넣으면 자동으로 사용됩니다.

## 파일 규칙
- 위치: `public/logos/{slug}.png`
- 슬러그는 `slugFor()` 함수 규칙: 소문자, 한글 그대로, 공백·괄호·특수문자는 하이픈
- 예시:
  - `래미안` → `public/logos/래미안.png`
  - `e편한세상` → `public/logos/e편한세상.png`
  - `린(LYNN)` → `public/logos/린.png`
  - `푸르지오 써밋` → `public/logos/푸르지오-써밋.png`

## 권장 사양
- 정사각 또는 와이드, 투명 배경 PNG
- 최소 512px
- 파일 없으면 브랜드 컬러 + 타이포그래피 폴백이 자동 표시됨

## 주의
브랜드 로고는 각 시공사의 상표권 자산입니다. 본인 학습용 외 공개 배포 시 권리자 확인 필요.
